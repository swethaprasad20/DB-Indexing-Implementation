1.Download the zip file
2.Unzip it
3.Export the opencsv jar file into classpath
	For MAC OS, command is 
	export CLASSPATH=$CLASSPATH:<path>/opencsv-2.4.jar
4.Compile the java file
	javac MyDatabase.java
5.Run the class file
	java MyDatabase


Indices are saved as binary file in indices.ndx file.
Binary file of CSV is saved in data.db
